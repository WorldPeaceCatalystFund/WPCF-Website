If you answered yes to any of those questions, you have come to the right place. Ghost Writer Alpha (GWA) is the first prototype of the Alethiometer. What is the Alethiometer? The Alethiometer is a device from the novel series The Golden Compass. In the novel, a user can pose a question to the Alethiometer by selecting three symbols. The Alethiometer then chirps back 3 symbols that represent an answer to the question. It is up to the user to interpret what the symbols mean as an answer. Similarly, with GWA, a user can pose a question by selecting a few symbols. GWA then returns an image stream which may hold an intelligable message which the user can interpret as an answer. In this way, GWA is a personal Alethiometer!

The images will sign you toward a meaningful answer. No special skills, training or abilities are needed. The answer should just come for most people. You already are familar with reading the signs, all you have to do is read GWA. What do you have to lose?

Want to test try it? Start with this question - What should I do for fun today? - this is a good general question that should bring an answer. Now think of 3-4 symbols you can use to pose the question. Having trouble? Try Dolphin, Compass and Wildman. Now hold the question and intention in mind and let GWA show you what you should do for fun. What images does GWA show? Does an answer form through signage from the images?
